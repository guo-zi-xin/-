**设计一个函数让两个数字相加，但不得使用`+`或者其他运算符**

```javascript
输入 a = 1 b = 2
输出 3
```

```typescript
const getSum = (a:number, b:number):number => {
 return [...Array(a), ...Array(b)].length
}
```

- **如果是多个参数求和**

```typescript

const getSums = (...args:number[]):number => {
  const arr = []
  for (let k = 0; k < args.length; k++) {
    arr.push(...Array(args[k]))
  }
  return arr.length
}
```
