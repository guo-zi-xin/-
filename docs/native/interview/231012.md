## 一些手写题目

1. **数组转树**

- 示例: 给定如下数组:

```javascript
    let source = [
      { id: 1, pid: 0, name: 'body', },
      { id: 2, pid: 1, name: 'title', },
      { id: 3, pid: 2, name: 'div', },
      { id: 4, pid: 0, name: 'html', },
      { id: 5, pid: 4, name: 'div', },
      { id: 6, pid: 5, name: 'span', },
      { id: 7, pid: 5, name: 'img', },
    ]
```

- 将其转换为下面这种形式

```javascript
let treeMap = [
    {
        id: 1,
        pid: 0,
        name: 'body',
        children: [
            {
                id: 2,
                pid: 1,
                name: 'title',
                children: [
                    {
                        id: 3,
                        pid: 2,
                        name: 'div'
                    }
                ]
            }
        ]
    },
    {
        id: 4,
        pid: 0,
        name: 'html',
        children: [
            {
                id: 5,
                pid: 4,
                name: 'div',
                children: [
                    {
                        id: 7,
                        pid: 5,
                        name: 'img'
                    }
                ]
            }
        ]
    }
];
```
